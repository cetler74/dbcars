## Understanding the Metrics

### Revenue Calculation
Revenue includes bookings with these statuses:
- ‚úÖ **Confirmed** - Booked and paid
- ‚úÖ **Active** - Currently renting
- ‚úÖ **Completed** - Rental finished

Revenue **excludes**:
- ‚ùå **Pending** - Not yet confirmed
- ‚ùå **Cancelled** - Cancelled bookings

### Booking Counts
- **Total Bookings**: All bookings regardless of status
- **Monthly Bookings**: Bookings created this month
- **Confirmed/Active/Completed**: Status-specific counts

## Common Questions

### Q: Why do some statistics show 0?
**A:** This is normal for new systems or if you have limited data. Statistics will populate as you:
- Create more bookings
- Add vehicles
- Publish blog posts
- Add extras

### Q: Why doesn't my test booking show in revenue?
**A:** Only bookings with status "confirmed", "active", or "completed" count toward revenue. Pending and cancelled bookings are excluded.

### Q: Can I export this data?
**A:** Currently, the page displays data visually. For exports, you can:
- Take screenshots of the statistics
- Use browser print function (Ctrl+P / Cmd+P)
- Export data from individual sections (bookings, customers, etc.)

### Q: How far back does "All Time" go?
**A:** "All Time" includes all historical data in your database from the first booking ever made.
