# Car Brand Logos

Place your car brand logo images in this directory with the following filenames:

## Required Logo Files:

1. **bugatti.png** (or .jpg, .svg, .webp)
2. **pagani.png** (or .jpg, .svg, .webp)
3. **koenigsegg.png** (or .jpg, .svg, .webp)
4. **mclaren.png** (or .jpg, .svg, .webp)
5. **rolls-royce.png** (or .jpg, .svg, .webp)
6. **lamborghini.png** (or .jpg, .svg, .webp)
7. **ferrari.png** (or .jpg, .svg, .webp)
8. **mercedes-benz.png** (or .jpg, .svg, .webp)
9. **aston-martin.png** (or .jpg, .svg, .webp)

## Image Requirements:

- **Format**: PNG, JPG, SVG, or WebP
- **Recommended size**: 200-400px width, transparent background preferred
- **Aspect ratio**: Flexible, but logos should be horizontally oriented
- **Background**: Transparent (PNG with alpha) or black background works best

## How to Add Logos:

1. Download or create logo images for each brand
2. Name them exactly as listed above (case-sensitive)
3. Place them in this directory: `/frontend/public/logos/`
4. The images will automatically appear on the homepage marquee section

## Notes:

- If an image file is missing, the brand name will be displayed as text instead
- The logos will automatically scale to fit the marquee section
- Supported formats: `.png`, `.jpg`, `.jpeg`, `.svg`, `.webp`
