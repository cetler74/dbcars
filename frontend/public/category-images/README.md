# Category Images

This folder contains the category images for the "Search by Category" section on the homepage.

## Required Images

Please add the following images to this folder:

1. **luxury-sedans.jpg** - Image of a luxury sedan (e.g., Mercedes-Benz S-Class)
2. **sports-cars.jpg** - Image of a sports car
3. **suvs.jpg** - Image of an SUV
4. **supercars.jpg** - Image of a supercar (e.g., Lamborghini Hurac√°n)

## Image Specifications

- **Format**: JPG or PNG
- **Recommended Size**: 800x800px or larger (square format works best)
- **Aspect Ratio**: 1:1 (square) is recommended for best display
- **File Size**: Optimize images to keep file sizes reasonable (< 500KB each)

## How to Add Images

1. Save your images with the exact filenames listed above
2. Place them in this folder: `/dbcars/frontend/public/category-images/`
3. The images will automatically be available at `/category-images/[filename]`

## Notes

- Images should have a dark background or be suitable for text overlay
- The code applies a dark overlay (50% opacity) to ensure text readability
- Images are displayed with `background-size: cover` to fill the entire card
